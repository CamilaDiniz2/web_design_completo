CSS: Modificando um arquivo HTML

Esta página contém as primeiras configurações para o uso de CSS.

CSS significa Cascading Style Sheet, é uma linguagem de estilo. Com sua utilização podemos adicionar cores, mudar tamanho de letras e estilos em um documento HTML.

    Com html pode-se modificar:

  • Cores (texto e fundo da tela)
  • Estilos e tamanho do texto
  • Margens e espaçamentos
